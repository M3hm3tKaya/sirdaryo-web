# Tech Context

## Stack
- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript 5+
- **Styling:** Tailwind CSS v4 (CSS-based config via @theme)
- **Animation:** GSAP 3.14 + ScrollTrigger, Framer Motion 12
- **Smooth Scroll:** Lenis 1.3
- **Icons:** Lucide React
- **Forms:** react-hook-form + zod
- **Fonts:** Inter, Space Grotesk, JetBrains Mono (via next/font/google)

## Key Differences (Tailwind v4)
- No tailwind.config.ts file
- Theme configured via `@theme inline` in globals.css
- Uses @tailwindcss/postcss instead of old postcss plugin
